BANK
------------

PHP 7.2.*
Yii 2 Basic

INSTALLATION
------------

You can then install this project using the following command:

~~~
composer install
~~~


Create db and write credentials in config/db.php 
~~~
 config/db.php
~~~

Run migrations
~~~
 php yii migrate
~~~

SCRIPT
------------
Deposits count for yesterday
~~~
 php yii calculation
~~~
Deposits count for exactly day
 ~~~
  php yii calculation 2020-02-29
 ~~~
